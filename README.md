# Прогнозирование успеваемости студентов с использованием нейронных сетей

## Описание

Этот проект посвящен изучению методов машинного обучения для решения задачи регрессии на наборе данных об успеваемости студентов. Цель проекта - научиться загружать, предобрабатывать данные, обучать различные архитектуры нейронных сетей, оценивать их производительность и сравнивать результаты.

## Данные

Данные для проекта доступны по ссылке (https://drive.google.com/file/d/1w6ZC_l0sQBCp5xh3wq1G4dFcB_DLHJc8/view?usp=drive_link). Необходимо скачать файл `student-por.csv`.

## Задачи

1.  **Загрузка данных:** Загрузить данные из файла `student-por.csv` и считать их в переменную pandas DataFrame.
2.  **Определение типа задачи:** Определить, является ли задача задачей классификации или регрессии. В данном случае, это задача регрессии, так как целевая переменная (G3 - итоговая оценка) является непрерывной.
3.  **Предобработка данных:**
    *   Разделить данные на тренировочную (80%) и тестовую (20%) выборки.
    *   Обработать пропуски (если есть) с использованием стратегий, описанных в семинаре №3 (отдельно для численных и категориальных признаков).
    *   Нормализовать численные переменные с помощью `StandardScaler` или `MinMaxScaler`.
    *   Закодировать категориальные признаки с помощью `OneHotEncoder`.
    *   Создать `Dataset` и `DataLoader` из библиотеки `torch`.
4.  **Обучение моделей:** Обучить три различные нейронные сети:
    *   Простая однослойная нейросеть с оптимизатором `SGD`.
    *   Нейросеть с одним скрытым слоем с оптимизатором `Adam`.
    *   Нейросеть с тремя и более скрытыми слоями с оптимизатором `Adam`.
5.  **Оценка моделей:**
    *   Посчитать `loss` на тренировочном и тестовом множествах для каждой эпохи и построить кривые обучения (learning curves) с помощью `matplotlib`.
    *   Для задачи регрессии посчитать метрику MAE (Mean Absolute Error) на тренировочном и тестовом множествах.
6.  **Сравнение результатов:** Сравнить метрики разных моделей и ответить на вопросы:
    *   Какая модель показала лучшие результаты?
    *   Наблюдается ли переобучение?
    *   Наблюдается ли недообучение?
    *   Как можно улучшить метрики моделей?
## Запуск
